# PracticumPrepProject
## מערכת לניהול נכסי נדל"ן - ארה"ב

## נושא הפרויקט
מערכת לניהול וחיפוש דירות למכירה ולהשכרה ברחבי ארצות הברית.

## מבנה מסד הנתונים (SQL Server)
המערכת מבוססת על 3 טבלאות עם קשרים ביניהן:
1. **Apartments** (טבלה ראשית): שומרת את פרטי הדירות (כותרת, תיאור, מחיר וכו').
2. **Cities** (טבלת עזר): רשימת ערים בארה"ב.
3. **Statuses** (טבלת עזר): ניהול מצב הדירה (למכירה, להשכרה, נמכר).

## פרוצדורות (Stored Procedures)
כל הגישה לנתונים מתבצעת אך ורק דרך הפרוצדורות הבאות:
* `GetAllApartments`: שליפת כל רשימת הדירות כולל אפשרות לחיפוש טקסט חופשי.
* `GetApartmentById`: שליפת פרטים מלאים על דירה ספציפית לפי מזהה.
* `CreateApartment`: הוספת דירה חדשה למערכת.
* `UpdateApartment`: עדכון נתונים של דירה קיימת.
* `DeleteApartment`: מחיקת רשומה מהמערכת.

## טכנולוגיות בפרויקט
* **Frontend:** Angular
* **Backend:** .NET Web API
* **Database:** SQL Server (Stored Procedures)
