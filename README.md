# PracticumPrepProject

# מערכת לניהול סוכנות תיווך נדל"ן - ארה"ב

## נושא הפרויקט
מערכת מקיפה לניהול נכסים, סוכנים ותהליכי מכירה/השכרה של נדל"ן ברחבי ארצות הברית.

## מבנה מסד הנתונים (SQL Server)
המערכת מבוססת על בסיס נתונים בשם `USA_RealEstate_Agency` הכולל 4 טבלאות מרכזיות:
1. **Apartments** (טבלה ראשית): ריכוז פרטי הנכסים, מחירים וקישורים לסוכנים וערים.
2. **Agents** (סוכנים): ניהול פרטי סוכני הנדל"ן המטפלים בנכסים.
3. **Cities** (ערים): רשימת ערים מרכזיות בארה"ב (פורמט: City, State).
4. **Statuses** (סטטוסים): ניהול מצב הנכס (למכירה, להשכרה, נמכר וכו').

## שכבת הנתונים (Stored Procedures)
כל הגישה לנתונים מתבצעת דרך פרוצדורות ייעודיות לשמירה על אבטחה וסדר:
* `GetAllApartments`: שליפת כל הנכסים כולל חיפוש טקסט חופשי ושיוך לסוכנים.
* `GetApartmentById`: שליפת מידע מלא על נכס ספציפי ופרטי התקשרות עם הסוכן.
* `CreateApartment`: הוספת נכס חדש למערכת ע"י סוכן.
* `UpdateApartment`: עדכון נתוני נכס קיים.
* `DeleteApartment`: הסרת נכס מהמערכת.

## טכנולוגיות
* **Database:** SQL Server
* **Server Side:** .NET Web API (C#)
* **Client Side:** Angular
